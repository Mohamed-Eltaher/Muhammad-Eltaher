== Description ==
WooCommerce Photo Review help you to allow your customer to post reviews include photos. With photo reviews, your store will have social proof, increase the trust of customers for product quality.

>[Try the Demo](http://new2new.com/?item=woocommerce-photo-reviews "Demo Photo Reviews for WooCommerce") | [Documents](http://docs.villatheme.com/?item=woocommerce-photo-reviews "Documents")

[youtube https://www.youtube.com/watch?v=zgT9qtCQIhQ]

- Allows customers to post reviews include photos

- Send email after successful purchase to ask for reviews

- Create coupon and send to customer after reviews are approved

### IMPORTANT NOTICE

- Plugin works based on WooCommerce plugin.

- It is released on WordPress.org and you can use plugin as free to build themes for sale.

### FEATURES

- **Reviews must include photos**: With this option, all reviews must include photos. You have an option to turn this off.

- **Photo review style**: Display image in customer's comments if any.

- **Review layout**: 2 layout to display reviews.

- **Photo options**: You can set the upload photo options as max photo size

- **Rating statistics**: Count and display review by rating for visual view and comparision

- **Filter Reviews**: Display reviews if they match the options: reviews contain images, reviews of verified owners, reviews grouped by number of stars rating.

- **Auto create and send coupon**: Create and send coupon for customers who post review

- **Coupon for photo review**: Choose to send coupon for only reviews include photos

- **Email design**: Design the email plugin will send customers to ask for review

- **Coupon settings**: Configure the coupons with fully set up as WooCommerce

- **Schedule time**: Select a time to send email to customers to ask for review after complete order

- **Customize email**: Design the emails send to customers with shortcodes

###PRO VERSION

- **All free version feature**

- **Multilingual:** Compatible with WPML and Polylang

- **Product info inside masonry popup:** Product title, average rating and add-to-cart button are displayed inside masonry popup

- **Add reviews at admin panel:** Admin can add reviews for all products

- **Filter reviews to moderate:** Admin can filter reviews by products, rating, with photos or not to easily moderate new reviews

- **Import reviews from Aliexpress:** You can import reviews from Aliexpress with options to select country, rating, with photos only...

- **Short code reviews:** Short code [wc_photo_reviews_shortcode] to display reviews of selected products/categories

- **Limit max files upload**:  Set maximum number of files that can be uploaded at a time.

- **Grid(Masonry) lay out**: Free to choose grid column number, grid color/background color, grid item color/bacground color, rating stars color.

- **Coupon for reviews of verified owner only**: For each completed order, customers will get one and only one coupon if they review a product contained in the order regardless of number of order line items.

- **Products restriction for coupon**: Restrict reviews of which products can customers get coupon.

- **Coupon usage restriction**: Restrict products that the coupon will be applied to.

- **Set Coupon code prefix**: Easier to look up.

- **Manually send review reminder**: Send manual review reminder for completed orders or even cancel a pending review reminder

- **Excluded product**: Select which products to be excluded in review reminder.

- **Customized "Review Now" button**: Change text color/background color of "Review Now" button in review reminder email.

- **Auto update:** New update notification when available.

- **Default active reviews tab:** Turn on to make reviews tab active on pageload
